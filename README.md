# remix-app

## Technologies used
* remix
* npm

## Getting Started
### Prerequisites
Make sure you have npm installed
You can check which bun version you have installed
```bash
npm --version
```

Install deps:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```


The website runs and can be accesed here http://localhost:3000
